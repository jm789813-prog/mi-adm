# 📊 Mi Admin - Gestor Personal Inteligente

Aplicación web moderna para gestionar tus datos personales de forma segura y eficiente.

## ✨ Características

✅ **Autenticación segura** - Login con validación
✅ **Escuela** - Gestiona tus materias y asignaturas
✅ **Finanzas** - Control total de tus gastos con cálculo automático
✅ **Pendientes** - Lista de tareas con checkboxes
✅ **Calendario** - Registro de eventos importantes
✅ **Diseño moderno** - Interfaz responsiva con Tailwind CSS
✅ **Almacenamiento local** - Todos tus datos guardados en localStorage
✅ **Animaciones suaves** - Transiciones y efectos visuales

## 🚀 Inicio Rápido

### 1. Clonar el repositorio
```bash
git clone https://github.com/jm789813-prog/mi-adm.git
cd mi-adm
```

### 2. Abrir en el navegador
Simplemente abre `index.html` en tu navegador web:
```bash
# En Windows
start index.html

# En macOS
open index.html

# En Linux
xdg-open index.html
```

## 💡 Cómo Usar

### Login
1. Ingresa las credenciales de demo
2. Haz clic en "Entrar"
3. Serás redirigido al dashboard

### Escuela 🎓
- Añade el nombre de la materia
- Haz clic en "Agregar"
- Elimina con el botón ❌

### Finanzas 💰
- Escribe el concepto de gasto
- Ingresa la cantidad
- El total se calcula automáticamente
- Visualiza todos tus gastos en tiempo real

### Pendientes ✅
- Escribe tu tarea
- Presiona Enter o haz clic en "Agregar"
- Marca como completada con el checkbox
- Elimina tareas completadas

### Calendario 📅
- Escribe el nombre del evento
- Selecciona la fecha
- Los eventos se ordenan automáticamente
- Los eventos más próximos aparecen primero

## 🔒 Seguridad

- ✅ Sesiones protegidas con localStorage
- ✅ Validación en todas las páginas
- ✅ Confirmación antes de cerrar sesión
- ✅ Redirección automática al login si no hay sesión

## 🎨 Tecnologías Utilizadas

- **HTML5** - Estructura semántica
- **CSS3** - Estilos modernos
- **Tailwind CSS** - Framework de utilidades
- **JavaScript Vanilla** - Lógica sin dependencias
- **localStorage API** - Almacenamiento local

## 📝 Notas

- Todos los datos se guardan en localStorage del navegador
- Los datos no se sincronizan entre navegadores o dispositivos
- Borrar el cache del navegador eliminará todos tus datos
